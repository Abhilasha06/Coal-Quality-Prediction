# Ensemble-Learning
Ensemble learning to identify three diffrent types of American coal.</br>
1st Model- cnn</br>
2nd Model- resnet50</br>
3rd Model- siamese network</br></br>


Test Accuracy -</br>
cnn- 88.20%</br>
resnet50- 96.82%</br>
siamese network- 95.19%</br>
weighted ensemble model-97.19%</br></br>
